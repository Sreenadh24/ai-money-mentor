# AI Money Mentor 💰

## Problem
Most people in India do not have proper financial planning.

## Solution
AI-based assistant that gives savings and investment advice.

## Features
- User inputs salary and expenses
- AI generates savings plan
- Multi-agent system (Input, Planning, Advice)

## Tech Stack
- Python
- Streamlit

## How to Run
streamlit run app.py

## Impact
If 100 users save ₹500/month → ₹50,000 total savings/month
